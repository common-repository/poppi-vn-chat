== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory (or install it directly from Wordpress app store).
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click "Settings" -> "Poppi Chat" and follow the instructions.
4. Go to poppi.vn, create a new account, or use your existing one
5. Add your unique account ID to the plugin.
You will need an Account ID, This can be found in your welcome email OR Contact our support team for assistance.
